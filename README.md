# SiteAgrinho

# 🌱 AgroTech - Projeto Agrinho 2024

## 📋 Sobre o Projeto
Site desenvolvido para o Projeto Agrinho, demonstrando como a Inteligência Artificial e a tecnologia estão revolucionando o agronegócio.

## 🚀 Tecnologias Utilizadas
- HTML5
- CSS3 (com variáveis CSS e animações avançadas)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts

## ✨ Funcionalidades
- Design responsivo
- Efeito de rastro do mouse
- Animações suaves
- Partículas flutuantes
- Contador animado
- Formulário de contato
- Efeitos de hover 3D
- Navegação suave
- Menu mobile
